# Desafios de Lógica em C#
Este repositório contém exercícios práticos de lógica de programação utilizando a linguagem C#. O objetivo é aprimorar a base algorítmica e a estruturação de projetos no ecossistema .NET.

## Tecnologias utilizadas
- C# (.NET)
- Lógica de programação
- Estruturação de projetos em classes e namespaces

## Desafios implementados
- **Password Validator (Validador de Senha):** Sistema de validação com regras de tamanho, caracteres especiais e dígitos.

## Organização
O projeto foi estruturado com foco na separação de responsabilidades, utilizando `namespaces` para organizar as classes de validação e manter o código limpo.
